\hypertarget{README.md_autotoc_md139}{}\doxysubsection{\texorpdfstring{Introduction}{Introduction}}\label{README.md_autotoc_md139}
\href{https://www.npmjs.com/package/i18next-browser-languagedetector}{\texttt{}}

This is an i18next language detection plugin used to detect user language in the browser, with support for\+:


\begin{DoxyItemize}
\item cookie (set cookie i18next=LANGUAGE)
\item session\+Storage (set key i18next\+Lng=LANGUAGE)
\item local\+Storage (set key i18next\+Lng=LANGUAGE)
\item navigator (set browser language)
\item querystring (append {\ttfamily ?lng=LANGUAGE} to URL)
\item html\+Tag (add html language tag \texorpdfstring{$<$}{<}html lang="{}\+LANGUAGE"{} ...)
\item path (\href{http://my.site.com/LANGUAGE/..}{\texttt{http\+://my.\+site.\+com/\+LANGUAGE/..}}.)
\item subdomain (\href{http://LANGUAGE.site.com/..}{\texttt{http\+://\+LANGUAGE.\+site.\+com/..}}.)
\end{DoxyItemize}\hypertarget{README.md_autotoc_md140}{}\doxysubsection{\texorpdfstring{Getting started}{Getting started}}\label{README.md_autotoc_md140}
